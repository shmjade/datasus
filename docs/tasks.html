  • Carregar distribuição horária de internações do SIH (query em refined) Query: SELECT EXTRACT(HOUR FROM dt_inter), COUNT(*) FROM trusted.internacoes GROUP BY 1. Usar como peso para random.choices().
  • Gerar evento JSON e publicar no tópico triagem-eventos Schema do evento: {"paciente_id": "uuid", "cnes_origem": "2226864", "timestamp": "ISO8601", "cid_suspeito": "I21", "fc": 110, "pas": 90, "fr": 22, "spo2": 94, "glasgow": 14}. Publicar com KafkaProducer(value_serializer=json.dumps.encode).
    kafka-python
  • Parâmetro --rate para controlar eventos/segundo (default: 1/s) Usar argparse. Modo turbo (--rate 10) para demo rápida.
📥 Consumer — motor de decisão pipelines/stream/consumer.py ▼
  • Consumir tópico triagem-eventos com KafkaConsumer(group_id='decisao-motor')
  • Classificar risco pelo Protocolo Manchester Vermelho: spo2 < 90 OR glasgow < 9 OR pas < 80. Laranja: fc > 130 OR fr > 30. Amarelo: demais alterações. Verde/Azul: sinais normais.
  • Se Vermelho: consultar refined.disponibilidade_leitos do município do cnes_origem Query: SELECT e.cnes, e.nome, r.total_leitos_sus FROM trusted.estabelecimentos e JOIN refined.disponibilidade_leitos r ON e.municipio = r.municipio WHERE r.total_leitos_sus > 0 ORDER BY r.total_leitos_sus DESC LIMIT 3.
